紫微鬥數基礎 ​

概覽 ​

紫微鬥數是研究 人 的心態和 外部環境 之間的關係的壹門學問，本站研究方向可能和大家理解的傳統紫微鬥數學習方向不大壹樣，所以如果妳以前對紫微鬥數已經有過研究和學習，在閱讀本站的觀點後可能會對妳以往學習過的知識產生壹些分歧，為了不影響妳的知識體係，請謹慎閱讀本站分享的知識。另外，本站所有分享均以科普掃盲為主，期望以最簡單易懂的方式讓大家了解紫微鬥數的趣味，旨在能讓大家了解紫微鬥數的運行原理。如有偏差，歡迎指正。

本係列文章為原創文章，轉載請標註出處。

陰陽五行 ​

在中國的道家思想裏，陰陽 和 五行 是基礎中的基礎，這裏我們不做展開，隻是讓大家有這個概念就好。

  • 陰陽

    道家的壹生二，生出來的就是陰陽，它代表事物的兩面，比如

    • 光 是 陽，影 就是 陰；
    • 雄 是 陽，雌 就是 陰；
    • 正電荷 是 陽，負電荷 就是 陰。

    所以，陰陽是壹個相對的概念，它代表了萬事萬物相輔相成，福禍相依，我們要找的，就是陰陽之間的平衡，避免過猶不及的情況產生。陽 會給人壹種比較強硬和熱烈的感覺，而 陰 給人的感覺通常是比較柔和和內斂的感覺。

  • 五行

    對於 元素 的概念，每個文化都有自己的詮釋，比如古希臘的 土、風、水、火 四元素

    古印度的 五元素，比古希臘的多了壹個 空，所以古印度的五元素分別是 土、風、水、火、空

    在中國的 五行 和其他學說的有相同的元素，也有不同的地方，中國的 五行 分別是 水、木、金、土、火，大家可能覺得金木水火土念起來比較順口，這裏主要是為了對應紫微鬥數裏的 五行局。

天幹地支 ​

在中國的傳統文化中，幹支紀年 是非常重要的基礎知識，它就像數學裏的加減乘除，英語裏的26字母壹樣。在紫微鬥數裏，整個星盤的排布就是按照 天幹 和 地支 來計算的。所以需要作為基礎熟記。

  • 十天幹

五行局 ​

五行局是紫微鬥數裏決定幾歲開始起運的依據，它來自於 六十甲子納音，但本站不討論 六十納音，如有興趣，請自行研究學習。五行局中間的數字，就是紫微鬥數裏大限起運的年齡。比如，水二局代表 2 歲起運，土五局代表 5 歲起運，關於五行局，大家知道這點就夠用了。五行局壹共五種，分別是：

  • 🌊 水二局
  • 🪵 木三局
  • 🪙 金四局
  • ⛰️ 土五局
  • 🔥 火六局

術語解釋 ​

在紫微鬥數裏，有很多“專業術語”，妳如果能夠對這些術語有壹些了解，在聽到別人解盤的時候就不會壹頭霧水了。聽多了就可以嘗試自己進行解盤，對星盤的研究是壹件很有意思的事情，希望大家都能體會到它的樂趣。下面我們壹壹解釋紫微鬥數裏常用的術語：

性別陰陽 ​

在紫微鬥數裏，會經常聽到 陽男陰女，陰男陽女的說法。這是指出生那年的 天幹 陰陽屬性，加上性別，得出4的種組合：

  • 陽男
  • 陽女
  • 陰男
  • 陰女

性別以及性別的陰陽屬性會影響某些結果的排列順序。

順行/逆行 ​

在紫微鬥數裏，順行指的是按 順時針 方向行進；逆行（或者逆部），是指按照 逆時針 方向行進。當妳聽到 陽男陰女順行 的時候就知道是要求妳用某個排列按照 順時針 方向排列，而 陰男陽女逆行(或逆部) 就是用某個排列按照 逆時針 方向排列。

星盤 ​

詳細的星盤知識請查看 星盤介紹。

  • 本命盤

    又叫 天盤，通過出生日期和時辰計算出來的那張盤，這張盤是固定的。

  • 大限盤

    又叫 地盤，天盤的十二個宮位每壹個宮位會輪流當值，每壹個宮位當值 十年，其中 命宮 是 第壹大限，後面的大限叫 第n大限。

  • 流年盤

    又叫 人盤，大限是十年輪換壹次，流年是 每年 輪換壹次。當年的 地支 所在的宮位就是流年命宮的位置。

  • 天地人三才

    是 本命盤、大限盤、流年盤 的統稱。

  • 格局

    某些 星耀 組合在壹起，會形成某種 格局，而格局的出現，就類似於遊戲裏面的裝備具有 套裝屬性 壹樣，會對某些屬性有影響。

宮位 ​

宮位的基本知識會在 宮位係統 裏面會詳細介紹。

  • 宮幹

    指宮位的 天幹 。

  • 本宮

    指目標宮位，比如查看 夫妻宮，那夫妻宮就叫 本宮。

  • 對宮

    目標宮位對角線上的宮位，又叫 四正位。

  • 三合位

    又叫 三才方、三方，是隻 本宮 和本宮的序號 加減4 所在的兩個位置。

  • 四正位

    又叫 對宮，是 本宮 序號 加6 所在的位置。

  • 三方四正

    三合位 加 四正位 的統稱。

  • 天羅地網

    指 辰、戌 兩個宮位，其中 辰 是天羅位，戌 是地網位。

  • 空宮

    指目標宮位內沒有 主星

  • 夾宮

    指目標宮位前後兩個宮位。

星耀 ​

紫微鬥數裏宮位中的重要組成部分，詳細介紹可以查看 星耀係統。

  • 流耀

    根據 大限 和 流年 而變化的星耀。

  • 遇、加、逢、同宮、同度

    指星耀在同壹個宮位裏面，比如說 紫微星 與 右弼星 同度，指這兩顆星在同壹個宮位裏面。

  • 會、照、沖、扶拱

    會 和 扶拱 是指星耀在目標宮位 三方位 裏，照 和 沖 是指星耀出現在 四正位，通常在解盤的時候會聽到 “與某某星會照”，就說明這顆星隻要出現在 三方四正 裏面就算。

  • 坐、鋸、守

    指星耀在某個宮位內。比如 貪狼 坐 夫妻宮，指的是貪狼星在夫妻宮裏面。

  • 雷門

    指 卯宮，因為卯宮位置在八卦的 震 位，所以叫 雷門。

  • 天門

    指 子宮。

  • 地門

    指 醜宮。
